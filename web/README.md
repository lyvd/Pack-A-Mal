# Web Interface of Package Analysis

## Installation

### Requirements
- Django
- requests

## Usage

```bash
cd Pack-A-Mal\web\package-analysis-web
python manage.py runserver
```

Connect to [127.0.0.1:8000/package-analysis](http://127.0.0.1:8000/package-analysis)
